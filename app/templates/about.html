About Ordiva To-Do App

This website was designed and developed by Lumia Qureshi using the Flask framework.

Ordiva is a simple yet powerful task management app that helps you organize your daily activities efficiently.

Features

  • Add, update, and delete your tasks easily.
  • Set due dates and times to stay on track.
  • Track task status: Pending, Working, Done.
  • User authentication for personalized task management.
  • Responsive and clean interface for all devices.
